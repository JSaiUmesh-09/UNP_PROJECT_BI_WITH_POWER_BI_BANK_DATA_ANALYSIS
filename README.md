# UNP_PROJECT_BI_WITH_POWER_BI_BANK_DATA_ANALYSIS

This project showcases a comprehensive Bank Data Analysis dashboard developed using Microsoft Power BI in collaboration with UNP Education, leveraging real-world banking data to generate meaningful insights. It involves data cleaning, transformation, and visualization to better understand customer behavior and financial patterns. The dashboard presents key metrics such as total transactions, average balances, and customer distribution across branches. It highlights insights related to account types, transaction trends, and demographic segments including age and gender. Interactive visuals enable in-depth analysis of branch performance, city-wise financial data, and customer engagement. The project also identifies key business challenges such as uneven cash flow and low-performing branches, along with actionable recommendations. Overall, it demonstrates strong skills in data analysis, business intelligence, and dashboard development using Power BI.
